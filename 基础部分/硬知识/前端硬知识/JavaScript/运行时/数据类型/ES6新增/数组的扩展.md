# 数组的扩展

## 数组常用的遍历方法
主要介绍forEach、map、filter、find、every、some、reduce，共同点是都不会改变原始数组。该部分将从每一种方法的特点以及常用场景进行分析：

+ 一、forEach：遍历数组

+ 二、map：将一个数组映射成另一个数组

+ 三、filter：过滤出符合条件的元素

+ 四、find/findIndex：查找符合条件的第一个元素/第一个元素索引

+ 五、every&some：查找数组中是否每一个元素都满足指定条件/是否有元素满足指定条件

+ 六、reduce：将数组合成一个值
